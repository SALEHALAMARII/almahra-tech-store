# Almahra Tech Store - Appwrite Ready

تم تعديل المشروع ليعمل بهذا المسار:

- Appwrite Auth لتسجيل دخول الأدمن.
- Appwrite Database لحفظ بيانات المنتجات.
- Appwrite Storage لرفع صور المنتجات مباشرة من الجوال أو الكمبيوتر.
- GitHub Pages لرفع واجهة المتجر.
- WhatsApp Checkout لإرسال الطلب.

## أهم ملف يجب تعديله

افتح:

`js/appwrite/config.js`

وضع:

- Project ID
- Database ID
- Products Table ID
- Images Bucket ID

راجع ملف:

`APPWRITE-SETUP.md`

## دخول الإدارة

افتح:

`admin/login.html`

تمت إضافة زر:

`الرجوع إلى الرئيسية`

في واجهة تسجيل الدخول.
